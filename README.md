# wangxuanwns.github.io

Personal academic homepage for Xuan Wang, built for GitHub Pages.

## Structure

- `index.html`: About Me
- `publications.html`: Publications and Education
- `style.css`: shared page styles

## Deploy

Push this repository to GitHub with the repository name `wangxuanwns.github.io`.
Once published with GitHub Pages, the site URL will be:

`https://wangxuanwns.github.io`
